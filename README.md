# trueOSweb

Marketing site for **trueOS** — Next.js (App Router), static export, deployed on Vercel.

## Scripts

- `npm run dev` — local dev server
- `npm run build` — production static export to `out/`
- `npm run start` — serve production build (after `next build` without static export, or use a static host for `out/`)
- `npm run lint` — TypeScript check (`tsc --noEmit`)

## Stack

Next.js 16, React 19, Tailwind CSS v4, minimal Radix (`dialog`, `slot`).
